# 🛒 Shop Management System

A simple Shop Management System built using Python (Flask) and SQLite database.

## 📌 Features
- Add new products
- Update product information
- Delete products
- Track stock availability
- Simple web interface using Flask

## 🛠️ Technologies Used
- Python
- Flask framework
- SQLite database
- HTML/CSS (for frontend)
